# nodejs-svc

This is a simple nodejs (typescript) service that starts and then ends.

## Usage to run locally

```bash
npm install
npm run build
npm start
```

## Usage to run in Docker

```bash
docker build -t nodejs-svc .
docker run nodejs-svc
```
